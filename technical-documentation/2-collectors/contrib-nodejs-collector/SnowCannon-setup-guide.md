### Overview

[Fluentd] [fluentd] is a lightweight log collector from the Treasure Data team. Fluentd supports a variety of different "output plugins", allowing SnowCannon to send SnowPlow events to Amazon S3 and other data sinks.

These installation instructions are adapted from the [Fluentd installation guide] [fluentd-install].
